# GitHub Profile Ricer (Antigravity Skill)

This repository contains an **Antigravity Skill** that allows the AI to automatically revamp and "rice" anyone's GitHub profile README with a premium, dynamic layout!

### Features of the Ricer:
- **Typing SVGs:** Dynamic typing effects for titles.
- **Skillicons:** Beautiful, unified tech stack icons.
- **Capsule Render:** Waving dynamic banners for the header and footer.
- **Tokyonight Analytics:** Seamless dark-mode GitHub stats, streak stats, and language charts.
- **Table Layouts:** Clean formatting for featured projects.

### How to Install (For AI Users)

If you are using Google Antigravity or a similar agent, you can install this skill easily:
1. Clone this repository or copy the `SKILL.md` file.
2. Place the `SKILL.md` file inside a new folder in your Antigravity skills directory: `~/.gemini/config/skills/github-profile-ricer/SKILL.md`
3. Antigravity will automatically load it. Next time you chat, simply ask: *"Hey, rice my github profile readme!"*

### How to Install (Manual Method - No AI needed!)

Don't have an AI assistant? You can still use this beautiful template! 
1. Open the `SKILL.md` file in this repository.
2. Scroll down to the `<!-- Header Section -->` where the HTML block begins.
3. Copy the entire HTML code block provided.
4. Go to your own GitHub profile repository (e.g. `github.com/your-username/your-username`) and edit your `README.md`.
5. Paste the code and simply replace the placeholders like `[NAME]`, `[GITHUB_USERNAME]`, and `[LINKEDIN]` with your actual details!
